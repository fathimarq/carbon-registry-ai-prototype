# 🌲 Carbon Registry AI Agent Training System

A prototype system for analyzing developer inquiries and preparing training data for an AI support agent in carbon markets.

#What This Project Does

This system demonstrates how to:
1. **Analyze developer support inquiries** to understand patterns and pain points

2. **Generate training data** for fine-tuning an AI
